# Lip Blushing in Delhi - Landing Page

A modern, mobile-first landing page for a lip blushing service in Delhi, built with React and Vite.

## Features

- Mobile-first responsive design
- Premium Korean aesthetic with soft colors
- Smooth scrolling and subtle animations
- WhatsApp and call integration
- Basic form validation
- SEO optimized with meta tags and JSON-LD schema

## Tech Stack

- React 18
- Vite
- Clean CSS (no frameworks)
- Mobile-first responsive design

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:5173


## Build for Production

```bash
npm run build
```

## Deployment

This project is ready for deployment on Vercel. Simply connect your GitHub repository to Vercel and deploy.
